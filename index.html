Features

聞き逃さない。だから、ついていける。

システム音声をそのまま翻訳

タブを共有するだけ。ヘッドフォンを使っていてもそのまま字幕化できます。

文字起こし・翻訳は無料

ブラウザ内のWhisper＋無料翻訳で処理。APIキーの登録は要りません。

API KEY 不要

整文／要約は必要な時だけ

無料APIで整形、またはコピーして普段のAIに要約を任せることもできます。

How it works

3ステップで、字幕が流れ出す。

STEP 01

始める

「始める」で本体へ移動します。

STEP 02

タブを共有

START → タブを選び「音声も共有」にチェック。

STEP 03

字幕＋整形

字幕が自動で流れ、「整える / 要約」でまとめられます。

利用にあたって（仕組み・使うもの）
音声認識はブラウザ内のWhisperで実行され、音声データは外部に送信されません。
翻訳は無料の翻訳エンジンを利用。整形・要約のみ任意でAPIを使います。
Chrome / Edge のデスクトップ版を推奨。Mac ではタブ音声のみ取得できます。

その場の会話に、ついていこう。

ブラウザを開いて「始める」を押すだけ。今すぐ字幕が流れ出します。
